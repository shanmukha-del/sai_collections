# Sai Collections — Luxury Artisan Jewellery

A premium e-commerce platform for Sai Collections, featuring a luxury storefront and a dedicated administrative management panel.

## Project Structure
This repository is split into two independent websites for separate deployment:

*   **/user**: The main storefront for customers. Features product browsing, category filtering, a shopping cart, and WhatsApp order integration with PDF invoice generation.
*   **/admin**: The management dashboard for product inventory, category management, and order tracking.

## Technology Stack
*   **Frontend**: Vanilla HTML5, CSS3 (Luxury Design System), JavaScript (ES6+)
*   **Frameworks**: React (via CDN), Framer Motion (Animations)
*   **Backend**: Supabase (Database, Storage, & Auth)
*   **PDF Generation**: jsPDF

## Deployment
Each folder should be deployed as a separate site on Netlify:
1.  Deploy the `user` folder as the storefront site.
2.  Deploy the `admin` folder as the administration site.
